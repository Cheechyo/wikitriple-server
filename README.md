# 기능
- 단어를 기준으로 만들어지는 문서 읽기
- 문서를 아무나 편집할 수 있는 기능 (문서는 마크다운 문법으로 작성할 수 있습니다)
- 회원가입하여 자기가 작성한 문서를 볼수 있게 함.
